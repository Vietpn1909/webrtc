## Cài Đặt & Chạy

### Yêu cầu
- Python 3.10
- Trình duyệt hỗ trợ WebRTC (Chrome, Firefox, Edge)

### Cài dependencies
```bash
pip install -r requirements.txt
```

### Khởi động server
```bash
python server.py
```
Mở `http://localhost:8080` trên hai tab trình duyệt, chọn User 1/2 rồi bắt đầu nói.
